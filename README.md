# CodeBundle

CodeBundle is a Streamlit-powered developer utility that allows you to **combine multiple code files or entire project folders into one clean, well-organized, copy-ready document**.  
It is designed especially for developers who want to share full project context with **AI models, collaborators, or code reviewers** easily and quickly.

---

## üöÄ Key Features

- **Upload folders or multiple files** directly from your system.
- **Automatic filtering** of unwanted directories like `__pycache__`, `node_modules`, `.git`, `.venv`, etc.
- **Smart file grouping** with clear path headers for readability.
- **Combined output generation** with all code in a single structured file.
- **Preview mode** to check your extracted files before generating.
- **One-click copy** to clipboard.
- **Downloadable bundled file** for later use.
- **File size detection** to alert you about large uploads.
- **Clean UI** built with Streamlit for smooth developer experience.

# üìö Usage Instructions

1. **Open the UI in your browser.**  
2. **Upload a folder or multiple files.**  
3. The system automatically **filters unwanted directories**.  
4. **Preview** the extracted files.  
5. Click **Generate Output** to combine them.  
6. **Copy** the combined file or **download** it.  
7. Use the generated file in **AI prompts, documentation, or code reviews**.

# üéØ Why CodeBundle?

Developers struggle to share complete project context with AI tools because:

- Uploading many files is slow  
- Manually merging files takes time

**CodeBundle solves this** by generating a single clean output file that includes:

- Proper **file separators**  
- Readable **formatting**  
- Unified **structure**  
- **AI-friendly content**  

Perfect for:

- LLM-based debugging  
- Code review  
- Documentation  
- Fast onboarding  
- Sharing project context in one click
